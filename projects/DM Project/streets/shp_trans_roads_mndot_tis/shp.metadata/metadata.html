Section 1 Overview
Originator Minnesota Department of Transportation (MnDOT)
Title Roads, Minnesota, 2012
Abstract This historic dataset represents road centerlines for all public roads within the state of Minnesota as of 2012. The roads are broken from intersection to intersection and attributed with information based on their designated route. Key attribute fields include route system (Interstate, US Highway, Minnesota Highway, County State Aid Highway, County Road, Township Road, etc.), Route Number (35W, 10, 53), and Name. A detailed description of the Roads layer attributes is included in Section 5 of this document - Entity and Attribute Overview.

Some route numbers are temporary. '900' Routes are for route segments that formerly were part of a trunk highway which was turned back to a local entity. These are temporary numbers assigned while MnDOT waits for an official local designation. These numbers are assigned in the 900-999 range and are not official route numbers but just for temporarily assigning data to unnumbered routes.

***The route IDs and measures contained in this data set are part of the old TIS system and are no longer supported at MnDOT. To download the current data set containing MnDOT's new LRS route IDs and more accurate measures, go to: https://gisdata.mn.gov/dataset/trans-roads-centerlines
Purpose This dataset was developed to fill a need at MnDOT for a continuous, statewide GIS base map of the transportation system. It was part of the old Transportation Information Sytem (TIS).

Section 2 Data Quality
Attribute Accuracy The best resource available at the time of the attribute updates was used. Ideally this was from the local municipalities or a construction plan but sometimes it came from a variety of different sources.
Logical Consistency Data have been topologically structured and verified.
Completeness The layer is complete for the entire state of Minnesota.
Horizontal Positional Accuracy Location accuracy for features derived from USGS 7.5-minute quadrangles cannot be assumed to exceed National Map Accuracy Standards for 1:24,000 scale maps (+/- 40 feet or 12 meters). Accuracy of updates based on imagery is dependent on the resolution of the imagery used. Imagery used for updates was the best imagery available at the time of the update. Features updated by MnDOT or local highway agencies have higher positional accuracy.
Vertical Positional Accuracy Not applicable
Lineage The original data source for the MnDOT Basemap Roads features was the USGS 1:24,000-scale USGS 7.5-minute quadrangle maps.

During the time period 1990-1995, the Roads and other MnDOT BaseMap data layers were digitized from USGS 7.5-minute quadrangles using MicroStation CADD. A total of 1745 quadrangles were digitized. For any given area, MnDOT used the most recent quadrangle available at that time. The digital MicroStation CADD files were converted from State Plane Coordinates to UTM and then converted into ARC/INFO coverages. The ARC/INFO files were merged into statewide, seamless coverages and then attributes were added.

Beginning in March, 1999, the Roads features were checked and updated, using 1997 digital orthophotos for Anoka, Carver, Dakota, Hennepin, Ramsey, Scott and Washington Counties and 1991 digital orthophotos for the remaining 80 counties.

The Roads features layer was updated continuously as information came in to the Office of Transportation System Management. The Minnesota Trunk Highway (TH), County State Aid System (CSAH), and Municipal State Aid System (MSAS) alignments and route data were improved each year through an annual status process. Changes to roads came from several sources including county engineers, city engineers, GIS personnel and the MnDOT State Aid office (which manages county state aid highways and municipal state aid streets), the most recent aerial photos available, and construction plans.
